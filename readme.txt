WooShare is a plugin that lets you add share links to the bottom of each of your posts. You can customize every link and add custom links easily!

== Description ==

WooShare lets you add share links to the bottom of each of your posts.
Every link has:
* Name (**required**)
* Image (optional)
* Text (optional)
* Title (optional)
* URL (**required**)

Every share link can include post meta such as title, author, etc., and a link to the post or blog.

== Installation ==

1. Upload `wooshare` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
